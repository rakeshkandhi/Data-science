# 01 — Python Fundamentals

Core Python skills required for AI/ML engineering.

## Concepts

| # | Folder | Topics |
|---|--------|--------|
| 01 | `01_basics/` | Data types, variables, strings, I/O |
| 02 | `02_conditional_statements/` | if/elif/else, match/case |
| 03 | `03_loops_and_functions/` | for/while, comprehensions, lambdas, decorators |
| 04 | `04_oop/` | Classes, inheritance, dunder methods, dataclasses |
| 05 | `05_libraries/` | NumPy, Pandas, Matplotlib, Regex |
| 06 | `06_web_scraping/` | requests, BeautifulSoup, Selenium basics |

## How to add a new concept

1. Create a subfolder: `07_new_concept/`
2. Add `concept_notes.ipynb` for theory + examples
3. Add `practice.py` for standalone scripts
