# 03 — Machine Learning

> ⚠️ **Note**: This module will be significantly expanded as we progress through the ML portion of the AI Engineer certification.

Core supervised and unsupervised ML algorithms, model evaluation, and sklearn workflows.

## Concepts

| # | Folder | Topics |
|---|--------|--------|
| 01 | `01_supervised_learning/` | Linear/Logistic Regression, KNN, SVM, Decision Trees |
| 02 | `02_unsupervised_learning/` | K-Means, DBSCAN, PCA, t-SNE |
| 03 | `03_model_evaluation/` | Cross-validation, metrics, confusion matrix, ROC-AUC |
| 04 | `04_feature_engineering/` | Encoding, scaling, imputation, feature selection |
| 05 | `05_ensemble_methods/` | Random Forests, Gradient Boosting, XGBoost, LightGBM |
| 06 | `06_sklearn_pipelines/` | sklearn Pipeline, ColumnTransformer, GridSearchCV |
