# Projects

End-to-end capstone and mini-projects applying concepts from all modules.

## How to add a project

1. Create a subfolder: `02_your_project_name/`
2. Add a `README.md` describing the problem, approach, and results
3. Add notebooks and scripts — they can import from `utils/`
